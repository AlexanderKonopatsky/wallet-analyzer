# DeBank Protocol Parser

Простой Python-скрипт для извлечения типа протокола (Protocol) кошелька с сайта DeBank.

## Возможности

- Получение типа протокола для любого адреса кошелька
- Извлечение баланса кошелька
- Работает с динамически загружаемыми страницами DeBank
- Headless режим (работает в фоне без открытия окна браузера)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Установите браузеры для Playwright:
```bash
playwright install chromium
```

## Использование

### Из командной строки

```bash
python debank_parser.py 0x6bfce69d1df30fd2b2c8e478edec9daa643ae3b8
```

### В коде Python

```python
from debank_parser import get_protocol_type

# Получить информацию о кошельке
result = get_protocol_type("0x6bfce69d1df30fd2b2c8e478edec9daa643ae3b8")

print(f"Протокол: {result['protocol']}")
print(f"Баланс: {result['balance']}")
```

## Структура результата

Функция `get_protocol_type()` возвращает словарь:

```python
{
    'address': '0x6bfce69d1df30fd2b2c8e478edec9daa643ae3b8',
    'protocol': 'Clipper',          # Тип протокола или None
    'balance': '$42,839',            # Баланс или None
    'success': True,                 # Успешно ли выполнен запрос
    'error': None                    # Описание ошибки или None
}
```

## Требования

- Python 3.7+
- playwright

## Ограничения

- Скрипт зависит от структуры HTML страницы DeBank
- При изменении дизайна сайта может потребоваться обновление селекторов
- Время выполнения ~5-10 секунд на один запрос (из-за загрузки страницы)
